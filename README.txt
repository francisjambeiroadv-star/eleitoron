TRIAGEM JURÍDICA INTELIGENTE - FRANCIS JAMBEIRO
Versão 1.0

COMO USAR:
1. Extraia o arquivo ZIP.
2. Abra o arquivo index.html no navegador.
3. Escolha a área jurídica.
4. Preencha os dados.
5. Clique em "Compartilhar link do sistema" para enviar o link ao cliente pelo WhatsApp.
6. O cliente preenche e clica em "Enviar resumo pelo WhatsApp".
